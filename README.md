# Ujjwal Singh — Portfolio

Personal portfolio website for Ujjwal Singh, Software Development Engineer & Founder of Qodebench Technologies.

## Stack

- Pure HTML, CSS, JavaScript — zero dependencies, zero build step
- Fonts: Space Grotesk + DM Sans via Google Fonts
- Animations: Intersection Observer API (scroll reveal), rAF loop (skew/stretch on scroll), CSS keyframes (marquee, hero slide-up)

## Running locally

No build step needed. Just open `index.html` in any browser:

```bash
open index.html
```

Or serve it with any static file server:

```bash
npx serve .
# or
python3 -m http.server 8080
```

## Deploying

Drop the folder contents into any static host:

- **GitHub Pages** — push to a repo, enable Pages from `main` branch root
- **Vercel** — `vercel --prod` from the folder
- **Netlify** — drag and drop the folder into the Netlify dashboard
